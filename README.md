## Development

### Required Software

- [direnv](https://direnv.net)
- [git](https://git-scm.com/)
- [node](https://nodejs.org/en/download/)
- [pre-commit](https://pre-commit.com/#install)

### Getting Started

**Setup**

```sh
$ direnv allow
$ pre-commit install
$ npm install
```

**Watch & Serve**

```sh
$ npm run dev
```

```sh
$ open http://localhost:1234/
```

### Build

```sh
$ npm run build
```
